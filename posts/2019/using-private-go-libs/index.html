Using private Golang libs

December 29, 2019

  • go
  • private lib

1 - Configure GIT (~/.gitconfig)

[url "ssh://git@github.com/"]
	insteadOf = https://github.com/

2 - Add the private repository location in GOPRIVATE env var

go env -w GOPRIVATE="github.com/<org>/<project>"
