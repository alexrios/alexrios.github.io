My first book on System Programming

28 June 2024 / 1 min read
  • book
  • system programming
  • go

Hey everyone, today is a special day! 🤓🎉

I’m thrilled to finally share with you the result of a project I’ve been working on for months: my first book on System Programming.

Writing this book has been an incredible journey. I’ve tried to put into it much of what I’ve learned over the years, in the clearest and most accessible way possible. If you’ve been following me, you’ll recognize my style here.

This book is intended to be a practical guide for anyone who wants to dive into the world of systems programming without any fluff and with real-world examples. If you, like me, believe that you can build amazing projects without major complications, this book is for you!

I’m very happy and eager to hear what you think.
